# FreshMart - AWS 3-Tier Grocery Application

## Architecture
- Frontend: S3 + CloudFront
- Backend: EC2 + Node.js
- Database: RDS MySQL

## AWS Services Used (10 Services)
1. VPC
2. IAM
3. EC2
4. RDS
5. S3
6. Security Groups
7. CloudWatch
8. SNS
9. CloudFront
10. Elastic IP

## Features
- Product listing
- Shopping cart
- User registration & login
- Order placement
